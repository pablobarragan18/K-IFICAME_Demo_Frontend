# 📊 K-LIFICAME — Demo Frontend

Demo visual interactiva del proyecto **K-LIFICAME**, una solución experimental orientada a la medición de satisfacción al cliente mediante dispositivos físicos basados en ESP32 y una plataforma web de análisis estadístico.

Esta versión fue desarrollada exclusivamente como demostración visual del sistema original, enfocándose en la experiencia del dashboard, visualización de métricas e identidad visual del proyecto.

---

## 🎯 Objetivo de la Demo

Mostrar:

- Interfaz del dashboard
- Visualización de estadísticas
- Diseño visual del sistema
- Experiencia de usuario
- Concepto general del proyecto

---

## 🧠 ¿Qué es K-LIFICAME?

K-LIFICAME fue un proyecto universitario desarrollado como prototipo funcional para negocios y establecimientos comerciales.

El sistema permitía a los clientes calificar la atención recibida mediante un dispositivo físico con botones basado en la escala de Likert.

Las respuestas eran enviadas a una plataforma web donde posteriormente se generaban:

- estadísticas
- gráficas
- métricas de satisfacción
- análisis de atención al cliente

---
